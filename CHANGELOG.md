# Changelog

## [0.0.1] - 2025-05-28
- Initial release.
- Create new Flutter projects with selectable platforms (Android, iOS, Web, Windows, macOS, Linux) from VS Code.
- Simple UI for project name, location, and platform selection.
- Runs `flutter create` with the correct `--platforms` option.
- Includes helpful README, license, and demo GIF.
